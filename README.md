## About

VFX GraphのCustom HLSL Blockを使用したサンプルです。
球上にpositionをセットするコンテキストノードをHLSLで作って、Initializeコンテキストで使用しています。

## Environment

|           | env             |
|:----------|:----------------|
| OS        | Windows 10 Home |
| Unity     | 2023.2.0a22     |
| VFX Graph | 16.0.2          |

## Usage

`Assets/CustomHLSLTestbed/Scenes/Main.unity`を開いて再生します。
